FAQ

What is Weave?

Weave is a lightweight universal mod loader for Minecraft that supports various clients across multiple platforms and versions.

How do I install Weave?

Simply download the latest version of Weave from the official website and run the installer. Follow the instructions and you're good to go!

Is Weave free?

Yes, Weave is completely free and open-source. You can check out the source code in our GitHub

How do I install mods?

Once you have installed Weave, you can download mods from the Weave discord and paste then in %userprofile%/.weave/mods. (Paste it in a run dialague to open the folder)
